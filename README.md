# spring-practice
공부한 것을 적용하는 프로젝트입니다. (기반은 [해당 강의](https://www.inflearn.com/course/%EC%8A%A4%ED%94%84%EB%A7%81%EB%B6%80%ED%8A%B8-JPA-%ED%99%9C%EC%9A%A9-1)에서 진행한 프로젝트에서 시작됩니다.)

## 🥦 Commit Type
- [FEAT] : 새로운 기능 추가
- [FIX] : 버그 수정
- [DOCS] : 문서 수정 (대체로 필요한 기록들을 Note.md에 기록할 때 사용)
- [STYLE] : 코드 변경이 없는 수정
- [REFACTOR] : 코드 리펙토링
- [TEST] : 테스트 코드
- [CHORE] : 빌드 수정
